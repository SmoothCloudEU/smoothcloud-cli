# SmoothCloud CLI

The control-panel of smoothcloud-controllers.

## Testing

```git clone https://github.com/SmoothCloudEU/smoothcloud-cli.git```

```go run .```

## Build

```git clone https://github.com/SmoothCloudEU/smoothcloud-cli.git```

### Windows

```go build -o smoothcloud.exe main.go```

```smoothcloud.exe```

### Linux

```go build -o smoothcloud main.go```

```./smoothcloud```
